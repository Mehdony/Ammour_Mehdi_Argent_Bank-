# ArgentBank

Welcome to ArgentBank! This project utilizes Vite React and Redux Toolkit to develop a banking application.

## Installation

Install/clone with :

```
git clone <https://github.com/Mehdony/Ammour_Mehdi_Argent_Bank-.git>
npm install

```

## Usage

Start with: 

```
npm run dev

```

## Features

- Login 
- Remember me
- Display name
- Edit name
